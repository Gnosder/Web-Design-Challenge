Summary: Lattitude vs X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To do this I builta dataset of 500 cities from the OpenWeatherMap API.

After creating the dataset, I used Matplotlib to plot various factors. I looked at the following factors: Temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of my analyis, as well as explanations and descriptions of any witnessed trends.
